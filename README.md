# 🚀 RT-ANGULAR-SAMPLES-FRONT

Este projeto é a interface de demonstração construída em **Angular 18** (Standalone Components & Signals) para interagir com o ecossistema de **Arquitetura Hexagonal**.

## 🏗️ Diferenciais Técnicos
- ✅ **Clean Interface:** Desenvolvida com foco em UX para orquestração de processos.
- ✅ **Reatividade Moderna:** Uso de **Angular Signals** para gerenciamento de estado.
- ✅ **Decoupled Architecture:** Comunicação via REST API seguindo contratos OpenAPI.
